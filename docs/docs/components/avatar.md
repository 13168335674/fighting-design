# Avatar 头像

## 基本使用

使用 `src` 属性设置图片的路径

`round` 属性可以配置圆角头像

<f-avatar src="https://tianyuhao.cn/images/my.jpg" />
<f-avatar round src="https://tianyuhao.cn/images/my.jpg" />

::: details 显示代码

```html
<f-avatar src="https://tianyuhao.cn/images/my.jpg" />
<f-avatar round src="https://tianyuhao.cn/images/my.jpg" />
```

:::
