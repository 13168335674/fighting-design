## 使用之前

在使用 `svg-icon` 组件之前，需要先进行安装

```shell
# pnpm
pnpm add @fighting-design/fighting-icon

# npm
npm i @fighting-design/fighting-icon

# yarn
yarn add @fighting-design/fighting-icon
```
